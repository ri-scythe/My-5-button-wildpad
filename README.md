# wildpad
A custom 5 button macropad built for the Hack Club Hackpad project. It features 2 layers for shortcuts and gifs, 5 keycaps, OLED screen and a rotary encoder. 

# CAD
Everything fits together using 4 M3 bolts and 4 hetset inserts. The case is made of 2 separate printed pieces.

# Firmware
This macropad uses [QMK](https://qmk.fm/) firmware.

- The rotary encoder changes volume.
- The 4 keys on the first layer are used to open apps and turn on bluetooth, the same 4 keys display different GIFS on OLED, and 1 of the keys is for changing the layer.

# BOM

- 5x Cherry MX Switches
- 5x DSA Keycaps
- 4x M3x5x4 Heatset inserts
- 4x M3x16mm Screws
- 1x EC11 Rotary Encoder
- 1x 0.91" OLED Display
- 1x Seeed Studio XIAO RP2040
- 5x SK6812 MINI LEDs
- 6x Through-hole 1N4148 Diodes
- 1x Case(2 separate parts)
